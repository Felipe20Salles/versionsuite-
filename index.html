Integração Redmine
Configure para importar tarefas no Kanban e enviar horas registradas. Requer o proxy local rodando.
Redmine → Administração → Enumerações → Atividades.
Para iniciar o proxy: copie redmine.env.example para .env, preencha as variáveis e rode node redmine-proxy.js.
Mapeamento de Status
Ao mover um card no Kanban, o status da issue no Redmine é atualizado automaticamente.
